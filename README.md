# 🌱 Urban Farming Backend API

A scalable **Urban Farming Platform Backend** built with:

- Node.js + Express
- TypeScript
- PostgreSQL + Prisma ORM
- Socket.io (Real-time plant tracking)
- JWT Authentication
- Docker & Docker Compose

---

# 🚀 Features

- 🔐 Authentication (JWT + Role-based access)
- 🌾 Farm rental system (location-based)
- 🛒 Organic produce marketplace
- 🌱 Real-time plant tracking (Socket.io)
- 💬 Community forum
- 🧾 Sustainability certification
- ⚡ Rate limiting + pagination
